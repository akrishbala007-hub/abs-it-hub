## üé® Keyword Placement Guide

### Page Title (Most Important!)
Format: `[Primary Keyword] | [Secondary Keyword] | [Brand]`

Example:
```
Refurbished Laptops Coimbatore | Quality Used Computers | ABS IT Hub
```

### Meta Description (155 characters)
Include:
- Primary keyword
- Location (Coimbatore)
- Call to action
- Unique selling point (warranty, affordable)

Example:
```
Buy quality refurbished laptops in Coimbatore at ABS IT Hub. Certified computers with warranty. Save up to 70%. Visit our Sowripalayam store today!
```

### H1 Heading (One per page)
Include primary keyword naturally

Examples:
- "Best Refurbished Laptops in Coimbatore"
- "Contact ABS IT Hub - Coimbatore's Trusted Computer Store"
- "Shop Quality Refurbished Desktops in Coimbatore"

### H2 Headings (Multiple per page)
Include secondary keywords

Examples:
- "Why Choose Refurbished Laptops?"
- "Our Sowripalayam Store Location"
- "Affordable Computers with Warranty"

### Body Content
- Use primary keyword 3-5 times per 1000 words
- Use secondary keywords 2-3 times
- Use variations naturally
- Don't force keywords (sounds unnatural)

### Image Alt Text
Include keywords in image descriptions

Examples:
- "Refurbished Dell laptop at ABS IT Hub Coimbatore"
